# v1.5 Conversation Decision Log

## D-001 — 릴리스 성격

**결정:** v1.5를 데이터 확장 버전이 아닌 `Visual Refinement & Interaction Recovery Release`로 정의한다.

**근거:** 사용자 요청은 신규 기체나 수치보다 카드, 검색, 레이더, 실척 비교, 포지션 맵, 차트 글자 개선에 집중됨.

## D-002 — 소스 우선순위

1. v1.4 실제 HTML/CSS/JS/JSON을 수정 기준으로 사용
2. v1.2 릴리스 보고서에서 UX 장점 추출
3. v1.3 릴리스 보고서에서 데이터·근거·미공개값 정책 유지
4. 사용자 화면 캡처를 시각 회귀 기준으로 사용

## D-003 — 포지션 맵 기본 레이블

**결정:** `대표+선택`을 기본값으로 사용한다.

**근거:** 50개 이상 기체명을 동시에 직접 표시하면 레이블 중첩을 구조적으로 피하기 어려움. 모든 점은 유지하고 세부 값은 툴팁과 클릭 상세로 제공함.

## D-004 — 포지션 맵 축

**결정:** `집중 보기`를 기본값으로 사용하고 0축 강제 포함을 해제한다. `전체 범위`는 별도 옵션으로 유지한다.

## D-005 — Radar 원값

**결정:** Radar 옆 중복 원값 표는 삭제한다. 원값은 상단 정밀 비교표와 Radar 포인트 툴팁에서 제공한다.

## D-006 — Radar 색상

**결정:** 저채도 파스텔 면 + 진한 외곽선 + 서로 다른 선형/마커를 병행한다.

## D-007 — 실척 비교

**결정:** 선택 기체 전체에 동일 px/m 축척을 적용하고 Overlay ON/OFF 모두 같은 축척을 사용한다.

## D-008 — 데이터 변경

**결정:** 항공기·출처 수치는 변경하지 않는다. 버전 메타데이터와 UI/QA 정책만 갱신한다.

## D-009 — 로컬 PDF 표기

**결정:** 현재 런타임에 PDF 원본이 없으므로 로컬 PDF 25개 포함 또는 해시 검증 완료로 표시하지 않는다. 외부 공식 원문 55건만 제공한다.
