# v1.5 Update Requirements Baseline

## 1. 요청 배경

v1.4의 통합 구조와 데이터 신뢰성은 유지하되, 실제 화면에서 확인된 카드·검색·레이더·실척 비교·포지션 맵의 가독성 문제를 v1.5에서 해결한다.

## 2. 원문 요구사항 정리

1. v1.2 기체 카드 형식이 충분히 반영되지 않음
2. 검색 시 검색한 항공기 이름이 HTML에서 겹쳐 잘 보이지 않음
3. v1.2의 다음 실척 형상 비교 구조 복원 필요
   - Top
   - Side
   - Front
   - Overlay ON/OFF
   - 크게 보기
   - SVG 저장
4. v1.4 좌석·페이로드 × 항속 포지션 맵의 글자가 겹치고 작으며 Plot이 한쪽으로 치우침
5. v1.2와 v1.3의 장점을 선별 반영
6. 다중 지표 정규화 6각 그래프가 너무 진하므로 v1.2와 같은 파스텔 계열로 개선
7. 6각 그래프 우측 중복 테이블 삭제
8. 6각 그래프 확대, 범례와 상호작용 고도화
9. Portal을 포함한 전체 그래프의 글자가 잘 보이도록 개선
10. 변경내용을 소스와 대화에 근거해 Revision MD로 체계적으로 관리
11. v1.5 전체 패키지를 ZIP으로 제공
