# v1.5 User Feedback Record

Date: 2026-07-14

## 사용자가 긍정적으로 평가한 항목

- v1.4 전체 방향
- v1.2의 기체 외형 실척 비율 구조
- Top / Side / Front 전환
- Overlay ON/OFF
- 크게 보기
- SVG 저장

## 사용자가 지적한 문제

- v1.2 카드 형식이 v1.4에 충분히 반영되지 않음
- 검색 결과 항공기 이름이 겹쳐 판독하기 어려움
- 포지션 맵 직접 레이블이 과도하게 겹침
- 포지션 맵 글자 크기가 작음
- 그래프 데이터가 한쪽으로 치우쳐 Plot 활용률이 낮음
- 정규화 6각 그래프 면 색상이 진해 여러 기체가 잘 구분되지 않음
- 6각 그래프 우측 원값 테이블이 상단 비교표와 중복됨
- Portal 등 전체 차트의 글자가 작음

## 설계 결정

- v1.2의 시각적 UX를 복원하되 v1.3의 데이터 규칙을 우선함
- 기본 포지션 레이블은 전체가 아니라 대표+선택으로 설정
- 전체 레이블은 옵션으로 유지하되 충돌 시 낮은 우선순위 항목 생략 가능
- 레이더 원값은 상단 정밀 비교표와 툴팁에서 제공
- 카드/검색/차트의 가독성을 기능 수보다 우선함
