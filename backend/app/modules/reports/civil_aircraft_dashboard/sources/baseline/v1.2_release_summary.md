# v1.2 Reference Summary for v1.5

v1.5에서 직접 참조한 v1.2의 장점은 다음과 같다.

- 큰 기체 SVG와 카드 중심 카탈로그
- 카드 내 핵심 사양을 한눈에 보는 구조
- 동일 기체 색상을 차트·표·SVG에 일관 적용
- 최대 6개 기체 비교
- 정규화 레이더 차트
- 좌석/페이로드–항속 포지션 맵의 직접 레이블과 Leader Line
- Top / Side / Front 실척 비교
- 공통 Length / Span / Height 축척
- Overlay ON/OFF
- 단일 기체 상세정보 강화

v1.5는 이 UX를 그대로 복제하지 않고, v1.3의 미공개값 비추정·필드 근거 정책과 결합해 재구현했다.
